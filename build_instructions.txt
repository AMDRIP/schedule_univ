# Инструкция по сборке установщика приложения

Это руководство поможет вам собрать дистрибутив (установщик) для вашего Electron-приложения на Windows, macOS или Linux.

## Шаг 1: Подготовка окружения

Перед началом сборки убедитесь, что у вас установлены:

1.  **Node.js и npm:** Если они не установлены, скачайте их с официального сайта [nodejs.org](https://nodejs.org/).
2.  **Зависимости проекта:** Откройте терминал или командную строку в папке с проектом и выполните команду для установки всех необходимых пакетов, включая инструменты для сборки:
    ```bash
    npm install
    ```

## Шаг 2: Настройка API-ключа Gemini

Приложение использует Gemini API для функции автоматического составления расписания. Для его работы **необходимо** установить API-ключ как переменную окружения.

**ВАЖНО:** Не вписывайте ключ напрямую в код! Приложение специально настроено на чтение ключа из окружения для безопасности.

### Для Windows (в Командной строке - cmd.exe)

Откройте командную строку и выполните команду, заменив `ВАШ_API_КЛЮЧ` на ваш реальный ключ:

```cmd
set API_KEY=ВАШ_API_КЛЮЧ
```

**Примечание:** Эта переменная будет установлена только для текущей сессии командной строки. Если вы закроете окно, ее нужно будет установить заново. Для постоянной установки используйте "Переменные среды" в настройках системы.

### Для macOS и Linux (в терминале)

Откройте терминал и выполните команду, заменив `ВАШ_API_КЛЮЧ` на ваш реальный ключ:

```bash
export API_KEY=ВАШ_API_КЛЮЧ
```

## Шаг 3: Сборка приложения

После установки зависимостей и настройки API-ключа можно приступать к сборке.

Откройте терминал или командную строку в папке проекта и выполните следующую команду:

```bash
npm run dist
```

Эта команда сначала автоматически соберет необходимые CSS-файлы, а затем запустит `electron-builder`, который создаст установочный файл в соответствии с вашей операционной системой.

## Шаг 4: Поиск готового установщика

После завершения процесса сборки (это может занять несколько минут) в корневой папке проекта появится новая папка `dist`. Внутри нее вы найдете готовый установочный файл:

*   **Для Windows:** `.exe` файл (например, `Система составления расписаний ВУЗа Setup 1.0.0.exe`).
*   **Для macOS:** `.dmg` файл.
*   **Для Linux:** `.AppImage` файл.

Этот файл можно передавать пользователям для установки приложения на их компьютеры.
